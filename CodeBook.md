# CodeBook for file result.txt

## Column subject

This column contains numerical ID, in range 1-30, of person with device.

## Column activity

This column contains activity performed by subject, it can be one of following:  
1. WALKING  
2. WALKING_UPSTAIRS  
3. WALKING_DOWNSTAIRS  
4. SITTING  
5. STANDING  
6. LAYING
